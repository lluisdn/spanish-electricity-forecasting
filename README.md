# Spanish Electricity Demand Forecasting

This project forecasts hourly electricity demand in Spain using historical demand data, calendar variables, weather features and machine learning models.

## Objective

Build an end-to-end forecasting project including:

- Data ingestion
- Feature engineering
- Baseline models
- Machine learning models
- Backtesting
- Model evaluation
- MLflow tracking
- FastAPI service
- Docker deployment

## Initial target

Forecast hourly electricity demand in Spain for the next 24 hours.
